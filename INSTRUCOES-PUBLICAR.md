# Como publicar o site da Astróloga Rosa no GitHub Pages

## O que você precisa
- Uma conta gratuita no GitHub (github.com)
- Os 3 arquivos desta pasta:
  - `index.html` — o site completo
  - `rosa-hero.jpg` — foto principal
  - `rosa-about.jpg` — foto da seção Sobre

---

## Passo a passo (15 minutos)

### 1. Criar conta no GitHub (se não tiver)
1. Acesse **github.com**
2. Clique em **Sign up**
3. Escolha um nome de usuário (ex: `astrologarosa`)
4. Confirme o e-mail

---

### 2. Criar o repositório
1. Após entrar, clique no **+** (canto superior direito) → **New repository**
2. Em **Repository name**, escreva exatamente: `astrologa-rosa`
3. Deixe marcado **Public**
4. Clique em **Create repository**

---

### 3. Enviar os arquivos
1. Na página do repositório recém-criado, clique em **uploading an existing file**
2. Arraste os 3 arquivos (`index.html`, `rosa-hero.jpg`, `rosa-about.jpg`) para a área indicada
3. Aguarde o upload dos 3 arquivos aparecerem na lista
4. Clique em **Commit changes** (botão verde no final da página)

---

### 4. Ativar o GitHub Pages
1. Clique na aba **Settings** (engrenagem)
2. No menu lateral esquerdo, clique em **Pages**
3. Em **Source**, selecione **Deploy from a branch**
4. Em **Branch**, selecione **main** e pasta **/ (root)**
5. Clique em **Save**

---

### 5. Acessar o site
Aguarde **2 a 5 minutos** e acesse:

```
https://SEU-USUARIO.github.io/astrologa-rosa
```

Substitua `SEU-USUARIO` pelo nome de usuário que você criou.

**Exemplo:** se o usuário for `astrologarosa`, o endereço será:
`https://astrologarosa.github.io/astrologa-rosa`

---

## Para anunciar no Google Ads

Use esse endereço como URL de destino nos anúncios.
O Google Ads aceita URLs do GitHub Pages normalmente.

Se quiser um domínio próprio no futuro (ex: `astrologarosa.com.br`):
- Registre no **registro.br** (~R$40/ano)
- Adicione o domínio em Settings > Pages > Custom domain
- O site continua hospedado no GitHub (gratuito)
